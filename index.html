API Choufli 7all

An open-source REST API providing quotes from the Tunisian series "شوفلي حل".

API Overview

This API provides random quotes from the iconic Tunisian TV series "Choufli Hal". It is 100% built in PHP using the Mabs micro framework.

Available Endpoints

GET /random

Returns a random quote in JSON format:

		Loading quote...
